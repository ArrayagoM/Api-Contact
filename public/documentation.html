Documentación

Guía completa para integrar y usar API Contact en tus proyectos.

Introducción

API Contact es un servicio que te permite recibir los datos de formularios de contacto directamente en tu email, sin necesidad de configurar un backend complejo.

1

Registro

Crea una cuenta con tu email y elige un identificador único (slug) que usarás para recibir tus formularios.

2

Integración

Modifica el atributo "action" de tu formulario HTML para que apunte a nuestro endpoint con tu slug personalizado.

3

Recepción

Cuando alguien complete tu formulario, los datos llegarán directamente a tu bandeja de entrada.

Endpoints

Endpoint de Formulario 
POST /form/:formSlug

Este es el endpoint principal al que debes enviar tus formularios. Reemplaza :formSlug con tu identificador único.

Ejemplo de Formulario HTML 
<form action="https://api-contact-one.vercel.app/form/tu-slug" method="POST">
  <input type="text" name="nombre" placeholder="Nombre" required>
  <input type="email" name="email" placeholder="Email" required>
  <textarea name="mensaje" placeholder="Mensaje" required></textarea>
  <button type="submit">Enviar</button>
</form>

Preguntas Frecuentes

¿Qué datos se almacenan en los servidores?

API Contact está diseñado con la privacidad en mente. Solo almacenamos:

  • Tu email (para enviarte los formularios)
  • Tu contraseña (encriptada)
  • Tu slug único

Los datos de los formularios que recibes NO se almacenan en nuestros servidores, solo se envían por email.

¿Puedo personalizar los campos del formulario?

Sí, puedes agregar cualquier campo adicional que necesites. Todos los datos del formulario serán incluidos en el email. 

<form action="https://api-contact-one.vercel.app/form/tu-slug" method="POST">
  <input type="text" name="nombre_completo" placeholder="Nombre completo">
  <input type="tel" name="telefono" placeholder="Teléfono">
  <select name="asunto">
    <option value="consulta">Consulta general</option>
    <option value="soporte">Soporte técnico</option>
  </select>
  <textarea name="mensaje"></textarea>
  <button type="submit">Enviar</button>
</form>
